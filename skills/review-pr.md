# Comprehensive PR Review Skill

**Skill Name:** `pr-review-toolkit:review-pr`
**Invoke:** `/review-pr`

## Description

Comprehensive PR review using specialized agents. This skill performs an in-depth review of pull requests with multiple analysis passes.

## When to Use

- For thorough PR reviews before merging
- When reviewing complex or large PRs
- To get detailed feedback on code changes
- For security-sensitive code reviews

## Capabilities

- Multi-agent review approach
- Security vulnerability scanning
- Performance analysis
- Code quality assessment
- Test coverage evaluation
- Architecture review
- Documentation check

## Example Usage

```
/review-pr
```

```
/review-pr 123
```

## Review Categories

### Security Review
- Injection vulnerabilities
- Authentication issues
- Data exposure risks
- Input validation

### Performance Review
- Algorithm complexity
- Resource usage
- Database queries
- Memory leaks

### Code Quality
- Readability
- Maintainability
- Code duplication
- Best practices

### Testing
- Test coverage
- Edge cases
- Integration tests
- Test quality
