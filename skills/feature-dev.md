# Feature Development Skill

**Skill Name:** `feature-dev:feature-dev`
**Invoke:** `/feature-dev`

## Description

Guided feature development with codebase understanding and architecture focus. This skill helps you build new features with a structured approach that considers the existing codebase patterns and architecture.

## When to Use

- Starting a new feature implementation
- When you want guided, step-by-step feature development
- Building features that need to integrate with existing architecture
- When you want Claude to understand your codebase before implementing

## Capabilities

- Analyzes existing codebase structure and patterns
- Provides architectural recommendations
- Guides through implementation steps
- Ensures consistency with existing code style
- Considers dependencies and integrations

## Example Usage

```
/feature-dev Add user authentication with JWT tokens
```

```
/feature-dev Implement a shopping cart feature
```

## Best Practices

1. Provide clear feature requirements
2. Mention any specific constraints or preferences
3. Reference existing patterns you want to follow
4. Specify integration points with existing code
