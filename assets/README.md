# Assets Folder

Add the following images before building:

## Required Files

| File | Size | Purpose |
|------|------|---------|
| `icon.png` | 1024x1024 | App icon |
| `adaptive-icon.png` | 1024x1024 | Android adaptive icon (foreground) |
| `splash.png` | 1284x2778 | Splash screen |
| `favicon.png` | 48x48 | Web favicon |

## Design Guidelines

- Use the CleanRecipe brand colors:
  - Sage: `#5C7C5A`
  - Terracotta: `#C4704E`
  - Cream: `#FAF8F5`

- Icon should be the leaf/spoon mark on cream background
- Splash screen should show logo centered on cream background

## Quick Placeholder Generation

You can use a tool like https://hotpot.ai/icon to generate these from a simple design.
